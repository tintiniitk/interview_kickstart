### Description
This coding problem is a part of the URL shortener system design problem.
The solution to that problem has a single microservice, which provides the following two functionalities
- Takes in a long URL and gives back a unique short URL, which is different each time even for the same long URL (encode)
- Takes in a short URL previously taken and return back the original long URL (decode)

The problem requires us to implment both encode and decode functionalities.
